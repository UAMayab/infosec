### OBJETIVO:
En este proyectto el objetivo es preparar el ambiente y las herramientas necesarias para llevar a cabo un caso de investigacion.
Toda la informacion necesaria para llevar a cabo este proyecto la encuentras detro del directorio: ** forensic_data ** de este repositorio de Github.

### ESPECIFICACIONES:
- Sigue las instrucciones detalladas en las diapositivas.
- A diferencia del Parcial 2, para el final tienes tres (3) diapositivas para ejecutar. Todas ellas se basan en las diapositivas del Parcial 2.
- Estudia y comprende las explicaciones detalladas en las diapositivas.
- Asegurate de comprender el proposito de cada activvidad, analizar los resultados y como cada uno de ellos abona a la investigacion.

### ENTREGABLE:
