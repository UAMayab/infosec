# Hack the Box Labs (HTB Labs)

## Registration
You need to register to the Hack the Box ([HTB](https://www.hackthebox.com/)) and create a Student accound using your Anahuac Mayab e-mail.
By creating a student account you will have access to solve a set of free labs that you will be working on along the semester. Optionally, 
you can buy Cubes to access pay labs.

## Weekly Labs
